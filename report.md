# 管理信息系统课程报告
## ——《兄弟服饰管理信息系统》 
#1 前言
##1.1  作业背景
本文根据《管理信息系统》课程要求而做。
 课程作业要求如下：

用管理信息系统开发工具（例如Visual Basic、Visual Foxpro、Access、PowerBuilder、Delphi等）开发一个实用的中小型管理信息系统。

1. 根据课程设计时间选择适当规模大小的设计课题。
2. 根据合理的进度安排，按照系统开发的流程及方法，踏实地开展课程设计活动。
3. 课程设计过程中，根据选题的具体需求，在开发各环节中撰写相关的技术文档，最后提交详细的课程设计报告，至少包括系统可行性分析报告、系统分析报告、系统设计报告三个主要部分。
4. 开发出可以运行的管理信息系统，通过上机检查。

##1.2  选题说明
###1.2.1公司介绍
徐州兄弟服饰有限公司由中国矿业大学管理学院学生创业项目成立。
专业从事徐州西服,徐州工作服定制以及徐州服装定制的生产和销售。公司生产各类高中档西服，男女职业装，风衣，个性文化衫以及演出服的租赁。公司的产品市场主要定位于各大国有企业商务正装、工作制服以及大学生求职正装。经过几年的努力，公司承袭了深厚的文化底蕴，并在产品的研发上不断创新，将时尚元素与传统正装完美结合。
为大学生量身打造职场正装，合身精致的正装，让广大学生在面试中更有自信，迈出成功第一步。
徐州兄弟服饰有限公司意在为每一人打造个性的专属服饰，包括个性文化衫、风衣等各种服装专业定制。
徐州兄弟服饰有限公司致力于为企业单位打造企业团体服饰，文化广告衫、企业工作服等。
###1.2.2公司问题
该公司在客户、供应商、计划单管理、销售管理、订单管理、库存管理、人员管理、工资管理等操作上都是原始的手工单据来传递信息，带来的一系列问题就显而易见了：工作的冗余和繁琐，信息的不准确，管理的疏忽，材料的浪费，决策依据的不真实等大大影响了公司的整体发展。而在供应商、销售客户的管理上现在公司还是一片空白，完全是依据管理层的经验来进行决策。任何关于公司客户、供应商的信息都是难以掌握和察觉的，造成了市场开拓的困难和物质供应的不透明性。
另外还有在实际管理中出现种种的漏洞，例如库存因为统计问题而造成流失，员工的工资因为疏忽而背离实际情况，这些都会给公司带来很大的损失。所以这种落后的管理方式已经不适合公司的长期发展，又尤其是在推进其市场化进程的阶段，面临着比较严峻的市场竞争的时候，企业更要团结一心，克服各种阻力，开发一套基于公司现状的信息管理系统便被提上了议事日程。

##1.3基本思路

###1.3.1系统价值
所谓管理信息系统是指一个由人和计算机等组成的，能进行信息收集、传输、加工、保存、维护和使用的系统，而管理信息系统的最终目的是为决策者服务，提供适合公司发展的战略计划。
管理的核心是决策，而管理信息系统可以辅助决策，特别是战略决策，它的好坏直接影响到企业的生存和发展。管理信息系统正在促使管理方法由定性向定性和定向相结合的方向发展。管理信息系统则可以结合管理需要，提供充分的数据，为模拟和制定预测创造一定的环境。使用计算机准确高效的处理方法和数据库紧凑的结构代替公司人员的手工操作和纸质传递，可以为企业节省人力，创造更多的价值，可以增加企业的竞争力。 

###1.3.2作业目标 

1. 系统开发的背景调查
2. 系统分析
3. 系统设计
4. 系统实现和系统维护

###1.3.3工作业绩

+ 小组主要工作

调查公司的运作情况，包括进货、销售、出租等经营活动，以及公司人员分工，部门职能等。对各环节进行分析讨论，思考如何将公司的各流程衔接起来，以及如何使公司的各环节实现系统化管理。

+ 主要收获

了解到公司各项流程，对各环节有一定把握，知道该如何着手进行系统设计和开发，目标形成初步定位。
